# Restaurant Landing Page

This is a landing page I created for a fictional premium restaurant.

The goal of this project was to practice building modern and responsive websites with a clean design, smooth navigation, and an enjoyable user experience.

## Live Demo

https://2cassstle.github.io/Restaurant-Landing/

## Features

* Responsive design
* Smooth scrolling navigation
* Menu section
* Chef section
* Gallery
* Customer reviews
* Reservation form
* Google Maps integration
* Modern UI

## Technologies

* HTML5
* CSS3
* JavaScript
